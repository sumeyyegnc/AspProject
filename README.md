# ☕ AspProject

This project was developed as part of the SoftITo Backend Development program. It is a dynamic ASP.NET Core MVC web application featuring a modern admin dashboard for managing products, cities, users, and reports. The application provides a clean and user-friendly interface while offering secure authentication and authorization using ASP.NET Core Identity.

The project includes a public landing page and a separate admin dashboard. Users can securely register, log in, manage their profiles, update their passwords, and access the dashboard after authentication. The dashboard allows administrators to efficiently manage application data through an organized and responsive interface.

## 🚀 Features

* Public Landing Page
* Secure Login & Registration
* ASP.NET Core Identity Authentication
* User Profile Management
* Change Password
* Product Management (CRUD)
* City Management (CRUD)
* User Management
* Reports Page
* Responsive Admin Dashboard
* Session Management
* Entity Framework Core Database Operations

## 🛠️ Tech Stack

* C#
* ASP.NET Core MVC
* Entity Framework Core
* Microsoft SQL Server
* ASP.NET Core Identity
* Bootstrap 5
* HTML5
* CSS3
* JavaScript
